
ФАЗА 1 : ЗАГРУЗКА КОНТЕНТА
1. SetupSourceFilesInValidOrderStep - просматриваем каталоги исходных данных, формируем список исходных файлов в правильном порядке
2. ReadSourceFileContentsStep		- считываем текстовый контент исходных файлов
3. ReadSourceXmlContentsStep		- парсим сожержимое файлов в виде XML с использованием XML/BXL
4. ResolveConditionalCompilation	- применяет условия компиляции к контенту
5. RegisterGeneratorsStep			- позволяет зарегистрировать классы генераторов, которые затем будут использоваться в заключительной фазе генерации
6. RegisterLinkTypesStep			- формирует полный список типов связей
>>> ВСЕ ИСХОДНЫЕ ДАННЫЕ ГОТОВЫ К ОБРАБОТКЕ

ФАЗА 2 : ОБРАБОТКА ГЛОБАЛЬНЫХ КОНСТАНТ И ГЕНЕРАТОРОВ

7. ExtractGlobalsStep				- извлекаем элементы "global", формируем словарь, отмечаем файлы, содержащие "~", как потенциальных носителей глобальных значений
8. ReIncludeGlobalsToGlobalsStep	- резольвит globals среди самих глобальных переменных (может быть отключена опцией no-reinclude-globals-to-globals)
9. EmbedGlobalsStep					- применяет значения констант во всех атрибутах и текстовых узлах исходных XML - файлов

>>> ГЛОБАЛЫ УБРАНЫ ИЗ ИСХОДНЫХ ДАННЫХ, ВСЕ ТЕКСТОВЫЕ УЗЛЫ И АТРИБУТЫ СОДЕРЖАТ ПОЛНОЦЕННЫЕ ЗНАЧЕНИЯ

ФАЗА 3 : ЭКСТРАКЦИЯ ОСНОВНЫХ ТИПОВ КОНТЕНТА, ПОДГОТОВКА ПЕРВИЧНЫХ СВОЙСТВ ТЕМ
10. ExtractSubsetsStep				- вытаскиваем и заносим в индекс все элементы subset
11. ExtractParametersStep			- вытаскиваем и заносим в индекс все элементы param и paramlib/param
12. ExtractThemasStep				- вытаскиваем все темы из всех файлов в context.Themas с резольвингом первичного наследования
13.ExtractThemaSelfItemsStep		- вытаскивает из тем элементы тем (в соответствии с настройками проекта)
14.ExtractThemaParametersStep		- наполняет таблицу атрибутов/свойств темы
15.ExtractThemaImportsStep			- заполняет список импорта тем
16.ExtractThemaExtensStep			- выполняет поиск и инициплизацию экстендеров
17.ExtractThemaLinksStep			- выполняет поиск и инициализацию связей

>>> ВСЕ ОСНОВНЫЕ ДАННЫЕ ПРОИНДЕКСИРОВАНЫ И РАЗДЕЛЕНЫ ПО РАЗДЕЛАМ, СОСТАВЛЕН ПЕРВИЧНЫЙ КАТАЛОГ ТЕМ, 
>>> ИСХОДНЫЕ ФАЙЛЫ ОЧИЩЕНЫ, В НИХ ОСТАЛИСЬ ТОЛЬКО ДОПОЛНИТЕЛЬНЫЕ ЭЛЕМЕНТЫ

ФАЗА 4 : РАЗРЕШЕНИЯ ИМПОРТА И ПАРАМЕТРОВ
18. ResolveParametersStep			- посредством наследования разрешает все параметры (без ембединга) во всей иерархии, включая генерики
19. CalculateParametersStep			- вычисляет итоговое значение параметра по подстановкам @name и "....${name}...."
20. CleanupParametersStep			- после вычищаем все пустые и дефолтные парамтеры - "","false","0"
21. RemoveNotUsedClustersStep		- очищает список тем от тем, не соответствующих целевому кластеру компиляции
21. EcoCleanupParametersStep		- (по умолчанию, опционален) вычищает параметры по принципу активности генерика по группам activeA, activeB, activeC
22. ResolveParametersToLinks		- выполняет подстановку параметров в связи

>>> ВСЕ ТЕМЫ НАСТРОЕНЫ ПО ПАРАМЕТРАМ И ИЗОЛИРОВАНЫ ДРУГ ОТ ДРУГА

ФАЗА 5 : РАЗРЕШЕНИЯ ЭЛЕМЕНТОВ
23. ResolveElementsImportStep		- правильно разрешает element, set, ext в коллекции ImportedXXXX
24. RemoveUnUsedThemasStep			- убирает все темы abst, generic, extend, inactive из набора - так как они уже ни для чего не могут использоваться
25. EcoCleanupElementsStep			- (по умолчанию, опционален) вычищает элементы и их расширения по принципу активности генерика по группам activeA, activeB, activeC
26. ResolveElementsStep				- осуществляет слияние item, set, ext
27. EmbedSubsetsStep				- разрешение колсетов (через addcols), опционально выдает варнинги при неотреферированных колсетах
28. EmbedItemParametersStep			- вставка параметров (ask/use)
29. EmbedParametersStep				- разрешение параметров темы во все дочерние элементы
30. CheckLibrariesStep				- проверяет, все ли библиотеки ссылаются на реальные элементы тех или иных тем
31. CallGeneratorsStep				- запускаем все элементы call CODE на выполнение во всех элементах темы

>>> ВЕСЬ КОНТЕНТ ГОТОВ К ГЕНЕРАЦИИ И СОХРАНЕНИЮ, ТЕМЫ  ГОТОВ К ЗАГРУЗКЕ

ФАЗА 6 : ЗАКЛЮЧИТЕЛЬНЫЕ МЕРОПРИЯТИЯ
32. GenerateXmlStep					- подготовка XML для тем
33. EmbedExtraDataStep				- вставка индексированных элементов из внешних данных
34. SaveXmlFilesStep				- сохраняет файлы на диске
